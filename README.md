# Currency Aggregator
Проект представляет собой агрегатор курсов валют с кэшированием и системой уведомлений.
## Основная идея

Сервис параллельно опрашивает несколько внешних API курсов валют:
- Центральный Банк РФ API

Полученные данные сохраняются в базу данных и кэшируются.
Пользователь может:
- получать актуальные курсы валют;
- просматривать историю изменений;
- строить графики;
- настраивать уведомления о достижении нужного курса.
