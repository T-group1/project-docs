# Архитектура проекта

Пользователь взаимодействует с Frontend.

Frontend отправляет запросы через API Gateway.

Gateway перенаправляет запросы в Rates Service.

Rates Service:

1. Запрашивает данные из Cache Service.
2. Если данных нет:
   - выполняет запросы к Frankfurter API;
   - выполняет запросы к API ЦБ РФ.
3. Сохраняет результат в БД.
4. Обновляет кэш.

Alert Service периодически проверяет курсы валют и уведомляет пользователей о достижении заданных значений.
